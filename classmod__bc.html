Detailed Description

Boundary condition module. 

- contains instantiations of all boundary condition for dynamically creating boundary conditions at run-time

Registering boundary conditions

  • To register a boundary condition: 1st: Import it's definition for use in the current module 2nd: Declare an instance of the boundary condition 3rd: Extend 'create_bc' to include a selection criteria for the boundary condition 4th: Under the selection criteria in 'create_bc', include a statement for dynamically allocating the boundary condition
