Return the value of a specified cylindrical duct mode.

For a specified azimuthal mode(m), and the eigenvalue of an associated radial mode(alpha), compute the value of the mode at radial location(r), bounded by the outer wall(rmax)

The mode values are normalized by N such that:

\( \int_0^1 N_{m \mu}^2 J(\alpha_{m \mu})^2 r dr = 1 \)

This is outlined in the document: "Fundamentals of Duct Acoustics" by Sjoerd W. Rienstra
