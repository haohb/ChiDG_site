ChiDG  0.1
A Chimera-based, discontinuous Galerkin solver
Installation

This page details the information required to successfully build the ChiDG project. One should be sure to first address the required dependencies of the project, which are detailed below. Also, a recent compiler version should be used. Tested compilers are listed below.

Obtaining ChiDG

ChiDG is hosted in a git repository on www.gitlab.com and may be cloned using either ssh or https protocols as

git clone git@gitlab.com:nwukie/ChiDG.git ChiDG
git clone https://gitlab.com/nwukie/ChiDG.git ChiDG

Dependencies

ChiDG depends on several libraries that enable IO and parallelization activities. These components are required to build ChiDG.

HDF5

HDF5 is used as a container for the ChiDG grid/solution files and is a required depencency of ChiDG.

HDF5 Website

MPICH

CMake

CMake is required to generate the 'make' configuration. Version 3.4.0 has been tested.

TecIO

Tecplot 360 is the primary visualization tool currently used for ChiDG. The TecIO libraries are distributed with the ChiDG project for multiple systems and should be linked accordingly without user intervention. If, for some reason, the packaged TecIO libraries are incompatible with your system, new libraries can be downloaded from the Tecplot website.

Compile

NOTE: ChiDG uses features of the most recent Fortran standard (F2008). Updated versions of Fortran compilers are required.

Successfully tested compiler versions:

  • gfortran 5.2

ChiDG uses the CMake build system. Out-of-source builds, as described here, are the only supported build method.

  • Change to the ChiDG root directory.
    cd ChiDG/
  • Create a new build directory.
    mkdir build
  • Change to the build directory.
    cd build
  • Execute the CMake command with options to configure the build
    cmake -DCMAKE_Fortran_COMPILER=gfortran-5.2 -DCMAKE_BUILD_TYPE=Release ..
  • Run make to build the ChiDG executable
    make
