Detailed Description

This equation set exists really just to test equationsets with more than one equation. The idea is just to compute the linear advecdtion solution twice at the same time. The equations are independent of each other. So, we can verify, for example, the volume flux jacobians for each equation. They should be the same as for the single LinearAdvection equation set.
