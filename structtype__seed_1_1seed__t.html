Detailed Description

Container that holds information on the element solution being linearized with respect to.

For example, if we were computing:

\( \frac{\partial F}{\partial Q_{idom,ielem}} \)

This container stores the indices of idom,ielem so the correct solution variables are initialized in the automatic differentiation.
