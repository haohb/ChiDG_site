Detailed Description

Module containing general constants that can be used consistently throughout the program.

Author
Nathan A. Wukie
Date
2/1/2016

These include:

  • Mathematical constants (pi, e, etc.)
  • Floating-point values for numbers and fractions (to ensure the correct floating point precision is used consistently)
  • Integers for consistent indexing
