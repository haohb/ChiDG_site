Detailed Description

Boundary condition module.

  • contains instantiations of all boundary condition for dynamically creating boundary conditions at run-time

Registering boundary conditions

  • To register a boundary condition: 1st: Import it's definition for use in the current module 2nd: In the 'register_bcs' routine, declare an instance of the boundary condition 3rd: In the 'register_bcs' routine, push an instanace of the bc to the registered_bcs vector
