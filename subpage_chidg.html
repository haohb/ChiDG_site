A Chimera-based, discontinuous Galerkin solver
ChiDG Instance
type(chidg_t) chidg

The chidg_t data type is the top-level interface for ChiDG. A chidg_t instance contains five primary components. These are:

type(chidg_data_t) data
type(timscheme_t) timescheme
type(nonlinear_solver_t) nonlinear_solver
type(linear_solver_t) linear_solver
type(preconditioner_t) preconditioner

Components

Data

type(chidg_data_t) data

The chidg_data_t data type contains all of the working information to run a chidg calculation. This includes:

type(mesh_t) mesh(:)
type(bcset_t) bcset(:)
type(equationset_wrapper_t) eqnset(:)
type(solverdata_t) sdata
