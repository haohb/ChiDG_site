Detailed Description

Class is used to manage the contributions and linearizations of functions.

Domain component contains arrays of logicals that indicate the status of a given function.

So, in this way, we can compute a flux on a face and add the contribution to both elements by registering that the function contribution was added. This container holds the registration information as arrays of logicals.
