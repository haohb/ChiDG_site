Detailed Description

Function for computing the eigenfunction for cylindrical duct modes. This could be passed to a root-finding routine to find the zeros, which are the cylindrical duct mode eigenvalues.

\( f(alpha_{mn}) = J'_m (alpha_{mn}) \)
