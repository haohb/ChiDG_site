ChiDG Instance
type(chidg_t) chidg

The chidg_t data type is the top-level interface for ChiDG. A chidg_t instance contains four primary components. These are:

  • type(chidg_data_t) data
  • type(timescheme_t) timescheme
  • type(matrixsolver_t) matrixsolver
  • type(preconditioner_t) preconditioner

Functionality

Required calls

A chidg_t instance requires a few

call chidg%init('env')
call chidg%init('finalize')
call chidg%init('chimera')
call chidg%close()

Components

Data

type(chidg_data_t) data

The chidg_data_t data type contains all of the working information to run a chidg calculation. This includes:

  • type(mesh_t) mesh(:)
  • type(bcset_t) bcset(:)
  • type(equationset_wrapper_t) eqnset(:)
  • type(solverdata_t) sdata
