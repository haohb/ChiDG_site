Usage

ChiDG Actions

chidg convert

To convert from block-structured Plot3D format grid files to ChiDG grid files, the convert action can be used as

chidg convert grid.x



chidg edit

To edit a ChiDG file(boundary conditions, domain info, etc.), the edit action can be used as

chidg edit grid_solution.h5

chidg post

To post-process a ChiDG solution into a visualization format, the post action can be used as

chidg post grid_solution.h5




Run ChiDG

Running chidg requires three things:

  1. The chidg executable
  2. A ChiDG-format grid file
  3. A chidg.nml namelist file

See Installation for direction on building the chidg executable.
See Generating a grid for direction on creating ChiDG-format grid files.
See Namelist Input for direction on a proper ChiDG namelist format.

In a directory with the ChiDG-format grid file and a valid chidg.nml namelist input file, the chidg simulation can be run by calling the ChiDG executable in that directory as

chidg
